Summary

通过对Test测试，发现Arrays.sort排序好像是以其中的年龄进行升序排序，其实不，是因为：UserComparator类的排序方法声明就是以User类中的Age字段进行排序UserComparator类，其实现的是接口：Comparator，该接口中的方法compare声明如下：int compare(T o1, T o2);

[接口中的方法默认为：public abstract修饰！]
